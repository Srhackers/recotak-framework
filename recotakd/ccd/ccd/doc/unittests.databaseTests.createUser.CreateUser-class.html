Class CreateUser

source code

Test checks for user creation functionality that is provided by the database.ccdUser module. A special focus is on input validation, hence this test neither explicitly tests the database.ccddb.create_user(..) nor the User.by_name/.byuid/.create_fromname/.foobar. Instead it calls atabase.ccdUser.new_user(..) which is called in case of an OP_NEWUSER operation.
