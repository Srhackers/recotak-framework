Function Details [hide private]

_send_plugins_output(sock, req_t, lock, fds_out)

  

Receive the plugins output, extract the rid and forward the content to
the concerning window. In addition, read the windows input to forward it
to the remote plugin.

input:
    channel_out the channel to write the plugins output to. In the end, the
                output is forwarded to the client
    req_t       the request triggerin the registration
    lock        the lock to access fds_out
    fds_out     the open file descriptors containing the plugins output
