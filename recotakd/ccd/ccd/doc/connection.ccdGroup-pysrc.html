 64          |            proxies             | 
 65          +--------------------------------+ 
 66   
 67   
 68   
 69      A connection group object has the following attributes: 
 70   
 71          grpid       the id of the connection group. an 8 byte hash 
 72          description a string to contain some descriptive wordings 
 73          chains      a list of chains ordered by priority. the priority is 
 74                      encoded in the list index. a chain with top priority has 
 75                      index 0. the higher the index, the lower the priority.
